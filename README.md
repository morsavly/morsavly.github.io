# morsavly

Mortgage & Savings Planner built as a static website.

## Features
- Mortgage calculator
- Savings planner

## Project structure
- `index.html` → app entry point
- `assets/` → icons and static files
- `js/` → application logic
- `styles/` → CSS files

## Deployment
This project is designed to be deployed as a static site on GitHub Pages.
